How do computers work?

Sun Nov 30, 2014

Using a computer doesnâ€™t require much -if any- understanding of the working of the inner machinery. However, just like knowing about the synergy drive of a hybrid car leads to a more energy efficient driver, understanding the relationship between hardware and software leads to more efficient programmers.

The digital computers we use have three main parts:

  • processing unit(s) (CPU),
  • memory unit(s), and
  • a control unit.

In addition to the above, a computer system has peripheral devices that can be grouped into three categories:

  • user interface devices,,
  • mass storage devices, and
  • communication devices.

Most of the computers today are based on the Von Neumann model of computing and works as follows:

- The memory stores all the computer programs, and all the data and values needed by those programs.

- A computer programs consist of step-by-step instructions which are executed in a logical sequence.

- The computer operates by causing the control unit to select an instruction from memory.

            - The instruction causes the control unit to get the relevant data from the memory and relay it to the CPU.

                        - The CPU then executes the program implied by the instruction.

                                    - The control unit saves the result in the memory and selects the next instruction.
